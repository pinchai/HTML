# Lesson 1: Introduction to Internet, WWW, HTML

## 1. What is the Internet?
- A global network connecting computers worldwide  
- Allows data communication (send & receive information)  
- Used for:
  - Email  
  - Social Media  
  - Websites  

---

## 2. What is WWW (World Wide Web)?
- A service running on the Internet  
- Accessed using a Web Browser  
- Uses:
  - URL → to locate websites  
  - HTTP/HTTPS → to transfer data  

---

## 3. Types of Websites

### Static Website
- Built with HTML, CSS  
- Content does not change  
- Simple and fast  

### Dynamic Website
- Connected to Database  
- Uses backend languages:
  - PHP, Python, Java, C#  
- Content changes based on user  

### Other Types
- E-commerce → Online shopping  
- Blog / News → Content sharing  

---

## 4. Static vs Dynamic

| Feature     | Static Website | Dynamic Website |
|------------|--------------|----------------|
| Technology | HTML, CSS   | Backend + DB   |
| Content    | Fixed        | Dynamic        |
| Speed      | Fast         | Slower         |
| Use Case   | Simple pages | Web apps       |

---

## 5. Web Browser
Popular browsers:
- Chrome  
- Edge  
- Firefox  
- Safari  

Functions:
- Open websites using URL  
- Render HTML/CSS  
- Execute JavaScript  

---

## 6. What is HTML?
HTML = HyperText Markup Language  

- Used to create web pages  
- Defines structure of content  

---

## 7. Example HTML Code
```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>
```

Save as `.html` → Open in browser
