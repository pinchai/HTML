# HTML Block and Inline Elements

## Block-level Elements

![Block
Elements](https://images.openai.com/static-rsc-4/VaOLXtJgwWlgeCfphEQbpmB0SOM7Fwigp-QE4gX1BQuudt_ir6gjO0M4LUIcPID8PK00-vcee8ARTW40CTpZHGO0A8HL1bSy6YeBHp5hvndQewmyqi12b3_k42Cu9mXrwSD9m5uEUvxlXXF-G27isVKOgVmxajorJ9AZNM4CvsNk3_b0uxHuxR6n5SFdDqfV?purpose=fullsize)

**Description:**\
Block-level elements always start on a new line and take up the full
width available.

**Common Block Elements:** - `<div>` - `<p>` - `<h1>` to `<h6>` -
`<ul>`, `<ol>`, `<li>` - `<section>`, `<article>`, `<header>`,
`<footer>`, `<nav>`

**Example:**

```html
<div>
  <h2>Title</h2>
  <p>This is a paragraph inside a div.</p>
</div>
```

## Inline Elements

![Block
Elements](https://images.openai.com/static-rsc-4/VaOLXtJgwWlgeCfphEQbpmB0SOM7Fwigp-QE4gX1BQuudt_ir6gjO0M4LUIcPID8PK00-vcee8ARTW40CTpZHGO0A8HL1bSy6YeBHp5hvndQewmyqi12b3_k42Cu9mXrwSD9m5uEUvxlXXF-G27isVKOgVmxajorJ9AZNM4CvsNk3_b0uxHuxR6n5SFdDqfV?purpose=fullsize)

**Description:**\
Inline elements do NOT start on a new line and only take up as much
width as necessary.

### Common Inline Elements:

**Text Formatting:** - `<span>` - `<strong>` - `<em>` - `<b>` - `<i>` -
`<u>` - `<small>` - `<mark>`

**Links and Media:** - `<a>` - `<img>` - `<br>`

**Form Elements:** - `<label>` - `<input>`

**Example:**

```html
<p>
  This is <strong>important</strong> text and <a href="#">this is a link</a>.
</p>
```

---

| Feature  | Block Element    | Inline Element       |
| -------- | ---------------- | -------------------- |
| New Line | Yes              | No                   |
| Width    | Full width       | Content width        |
| Usage    | Layout/Structure | Text/Content styling |
